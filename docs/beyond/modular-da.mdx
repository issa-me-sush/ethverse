# Modular DA Stack

## The Rise of Modular Blockchain Architecture

2023 saw the emergence of specialized data availability solutions:

- Celestia mainnet beta
- EigenDA launch
- Avail testnet
- DA sampling
- Data availability committees

## Key Components

### Data Availability Layer
- DA sampling
- Data attestation
- Fraud proofs
- Light client support

### Network Solutions
- Celestia
- EigenDA
- Avail
- Polygon Avail

### Technical Features
- Data sampling
- Reed-Solomon encoding
- KZG commitments
- Light client verification

## Implementation Approaches

### Celestia
- Minimal consensus
- DA sampling
- Namespace-based ordering
- Optimized for data

### EigenDA
- Restaking mechanism
- Operator network
- Distributed DA
- Economic security

## Impact on Scaling

The modular DA stack enables:
- Cheaper data availability
- Better scalability
- Specialized layers
- Rollup optimization

This represents a shift toward modular blockchain architecture, enabling more efficient and specialized scaling solutions.
